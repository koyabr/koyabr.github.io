在2020年尝试了第一次黑苹果，当时的配置如下：

  • 板U：AMD R5 2600 + 迫击炮 B450M
  • 显卡：AMD RX 580（后来嫌太吵换了单风扇的渣渣560D）
  • 存储：普通DDR4 2666、海康C2000（因为掉盘换了铠侠RC 500）

直接抄了一套网友的EFI，完全没研究细节，当时还是Clover引导，安装了MacOS 10.14，对无线没要求，蓝牙买了一个USB接收器搞定，不使用Airdrop等花里胡哨的苹果官方把戏，日常使用没有问题。

时间来到2022年618，冲动消费一把，板U升级到R5 5600 + 重炮手 B550M WiFi II，SSD换成了西数SN 770，显卡打算换6600但因为贫穷决定再等等。

这次没有现成的EFI可抄了，并在付出了一整个周末后发现主板的WiFi和蓝牙都换成了联发科，连Windows下都要去华硕官网下载驱动，MacOS下就别指望有大佬用爱发电写驱动了。痛定思痛，又花了一个周末研究了几套网友的EFI、仔细爬完了官方教程，最终的经验教训汇成下文。

Step-by-step guide

首先认真读一遍官方教程 https://dortania.github.io/OpenCore-Install-Guide/

制作USB启动盘

  • 用脚本下载官方镜像，或者依靠白苹果系统更新下载的系统镜像，从中找到名如 Install macOS xxx.app 的安装包
  • 将U盘格式化为HFS+文件系统和GUID分区表，假设命名为MyVolume
  • 安装包里自带了刷U盘的程序，执行 sudo /Applications/Install\ macOS\ Big\ Sur.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume

准备EFI所需文件

  • 下载OpenCore release包 https://github.com/acidanthera/OpenCorePkg/releases/
  • 收集Firmware Drivers，https://github.com/acidanthera/OcBinaryData/blob/master/Drivers/HfsPlus.efi 必须下载，性能比OpenHfsPlus.efi更好，前者是苹果私有实现，后者是开源实现
  • 收集Kexts

  • 收集SSDT
    • B550主板需要 https://github.com/dortania/Getting-Started-With-ACPI/blob/master/extra-files/compiled/SSDT-CPUR.aml
    • USB正常工作需要 https://github.com/dortania/Getting-Started-With-ACPI/blob/master/extra-files/compiled/SSDT-EC-USBX-DESKTOP.aml
  • 收集启动界面GUI的资源文件
    • https://dortania.github.io/OpenCore-Post-Install/cosmetic/gui.html#setting-up-opencore-s-gui

编写config.plist

  • 用OpenCore release里的sample.plist为基础来改造
  • 主要遵循教程 https://dortania.github.io/OpenCore-Install-Guide/AMD/zen.html#starting-point
  • 注意boot-args
    • alcid参数是为了声卡，11是Realtek常用的
    • revcpu=1配合RestrictEvents.kext，让CPU名称能正常显示在mac系统属性里
  • 为了启动界面GUI美观，注意UIScale参数
    • 在NVRAM的Add部分设置UIScale为02，Delete部分注意添加UIScale
    • 在UEFI的Output部分设置UIScale为-1，否则会报错“两个地方重复设置了该参数”

BIOS设置

disable

  • Fast Boot(快速启动)
  • CSM(兼容性支持模块)

enable

  • Above 4G decoding(大于 4G 地址空间解码)
  • EHCI/XHCI Hand-off(接手 EHCI/XHCI 控制)
  • OS type: other types(操作系统类型: 其他)

进系统后

  • 有线网卡需要手动设置速度为1000M，如果不生效，检查下路由器的LAN口相关配置
  • 无线别指望，蓝牙用外置USB
  • 睡眠偶尔会遇到无法唤醒的问题，只能强制关机，重启后会要求进入BIOS设置界面，还好BIOS并没有被重置
