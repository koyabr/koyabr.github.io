Some nice websites to learn programming

2020-09-11

Contents

  • 算法
  • 综合

虽然有CSDN为代表的一票拷贝狂魔，让网络上的中文编程资料质量令人一言难尽，但我们的眼睛总是可以发现一些美好的，比如这几个很不错的中文站点。

算法

  • 后浪中的巨浪，我大学要有这觉悟……小学弟的题解，特点是有很多框架性的提炼和总结，适合速成、突击、抱佛脚、前排仰慕
    • labuladong/fucking-algorithm: 刷算法全靠套路，认准 labuladong 就够了！English version supported! Crack LeetCode, not only how, but also why. (github.com)
  • The Missing Manual of Competitive Programming 来自Hulu员工的编程竞赛wiki，可能是该领域内容最全面的中文站点，有很多我这种门外汉闻所未闻的知识
    • OI Wiki (oi-wiki.org)

综合

  • 比github上一票学生和社畜的面试笔记仓库更系统化（颜值更高）的……笔记仓库
    • Java 全栈知识体系 (pdai.tech)
